Workspaces
Core

Build

The resume engine. Vector-based targeting with per-bullet include/exclude controls, skill group routing, Typst WASM PDF rendering with WYSIWYG preview. 8 theme presets with full token-level overrides. Density controls, presets with dirty-state tracking, and round-trippable JSON import/export.

Core

Pipeline

Full job tracking. Company, role, URL, compensation, status, notes, JD storage, vector linkage, and activity history. Filter by tier, status, and search. Built-in analytics. Hands off directly into Build analysis and Prep generation.

AI

Research

Infers a search profile from your resume data. Runs targeted searches with vector priorities, filters, and exclusions. Automatically skips companies you've rejected. Push results straight into Pipeline.

AI

Prep

Interview prep decks linked to pipeline entries. AI generates cards from the JD, your resume, and company research. Search, filter, duplicate, edit. Practice mode for rehearsal.

AI

Letters

Template-based cover letters with paragraph-level vector controls. AI drafts from pipeline context — opportunity, vector, company research, and assembled resume data. Reusable templates persist across sessions.

Core

Persistence & Backup

Encrypted workspace backups with passphrase-based WebCrypto. File System Access for native save/load. Sync status in the shell, backup reminders when local changes outpace saved state. Tenant-aware — ready for hosted backends.
